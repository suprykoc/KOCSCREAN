# Ekran Kaydedici Uygulaması (Screen Recorder Application)

### Açıklama / Description
Türkçe: Bilgisayar ekranınızdaki her şeyi kaydedin, webcam görüntünüzü izleyin ve kaydedin. Kullanıcı dostu arayüzü ile modern bir ekran kaydedici!

English: Record everything happening on your screen and toggle your webcam for an enhanced recording experience. A modern, user-friendly screen recorder!

---

### Özellikler / Features
- 📽️ Ekran Kaydı / Screen Recording
- 📸 Webcam Desteği / Webcam Support
- 🎨 Minimalist Arayüz / Minimalist Interface
- 🎮 Kullanım Senaryoları / Use Cases:
  - Online dersler, toplantılar, oyun içi kayıtlar / Online classes, meetings, in-game recordings.

---

### Kullanım / Usage
1. Uygulamayı çalıştırın / Run the application.
2. `Kaydı Başlat` butonuna tıklayın ve bir dosya adı seçin / Click `Start Recording` and choose a file name.
3. `Kaydı Durdur` ile kaydı geçici olarak durdurabilir, `Kaydı Bitir` ile tamamlayabilirsiniz / Pause with `Stop Recording` or finish with `End Recording`.

---
